# CayoPerico
CayoPerico is a resource for your FiveM server to easily add the Heist Island with Los Santos.\
Map and minimap are functional.
This resource was made with the work of [nta](https://forum.cfx.re/u/nta), [Blaxered](https://forum.cfx.re/u/Blaxered), [AvaN0x](https://forum.cfx.re/u/AvaN0x), [DocWeird](https://forum.cfx.re/u/DocWeird), [d0p3t](https://forum.cfx.re/u/d0p3t)\
Thanks to us, and their posts on Cfx.RE forum.

## Installation
Just download the source code, put it in your resources folder and make sure to start it in your server.cfg\
Eg : `ensure cayoperico`
